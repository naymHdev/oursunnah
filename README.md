# E-Commerce Website Project

## 🌟 Our Sunnah

This is a modern e-commerce website built with Next.js, TypeScript, and Tailwind CSS. The project provides a clean and responsive user interface with essential features like navigation, product display, testimonials, and a newsletter subscription.

---

## 📦 Folder Structure

```
src/
├── app/
│   └── page.tsx              # Main page component
├── components/
│   ├── Navbar.tsx            # Navigation bar component
│   ├── Hero.tsx              # Hero section component
│   ├── Collections.tsx       # Collections section component
│   ├── FeaturedProducts.tsx  # Featured products section component
│   ├── Testimonials.tsx      # Testimonials section component
│   ├── EditorialSection.tsx  # Editorial section component
│   └── Newsletter.tsx        # Newsletter subscription component
└── styles/
    └── globals.css            # Global CSS styles
```

---

## 🧩 Features

- **Responsive Design:** Built with Tailwind CSS for a responsive and mobile-friendly layout.
- **Navigation Bar:** A customizable navigation bar for easy site navigation.
- **Hero Section:** A visually appealing hero section with a call-to-action.
- **Collections Section:** Displays curated product collections.
- **Featured Products:** Highlights key products with images and descriptions.
- **Testimonials Section:** Displays customer reviews and feedback.
- **Newsletter Subscription:** Allows users to subscribe to the newsletter.
- **Footer:** Contains links, contact information, and social media icons.

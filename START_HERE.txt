🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

Problem: "pnpm-workspace.yaml not found"
→ Solution: Run script from repo root directory (where pnpm-workspace.yaml is)

Problem: Script can't find files
→ Solution: Make sure all 4 .tsx/.ts/.sh files are in repo root

Problem: Modal doesn't open
→ Solution: Clear cache: rm -rf apps/web/.next && pnpm dev

Problem: Search doesn't work
→ Solution: Check backend running: cd apps/api && npm run dev

Problem: Git push fails
→ Solution: Configure git: git config --global user.name "Name"

Problem: Styling looks wrong
→ Solution: Verify Tailwind colors in tailwind.config.js, rebuild with pnpm dev

For more help → See SETUP_INSTRUCTIONS.md
